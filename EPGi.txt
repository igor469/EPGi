1. Основная цель проекта
Основная цель проекта - реализация консольной (CLI) программы на Python, которая предоставляет пользователю интерактивные средства просмотра актуальной программы передач каналов IPTV.

2. Структура проекта
Структура проекта -
EPGi/
├── EPGi.py           ← основная программа
├── EPGi.ini          ← файл параметров
├── EPGi.log          ← файл журнала-протокола работы программы 
└── README.md

3. Ограничения и допущения проекта
Файл EPG представлен в структуре XML.
Файл EPG может быть архивирован.
Количество каналов IPTV в одном файле EPG: 1 - 10000.
Количество дней программы передач для одного канала: 1 - 14.

4. Структура, состав входного файла параметров EPGi.ini
В файле параметров EPGi.ini пользователем указывается 2 параметра:
url - http ссылка на актуальный файл EPG. 
page - Количество строк в одной странице вывода информации пользователю.
Пример файла EPGi.ini
-----
# Comment

[Main]  	# Главный-общий раздел параметров
url=http://epg.ott-play.com/edem/edem.xml.gz  # http ссылка на актуальный файл EPG
N=40    	# Количество строк в одной странице
-----

5. Файл журнала (EPGi.log)
В файл журнала (EPGi.log) выводится следующая информация:
- время старта программы
- все ошибки работы
Времена в сообщениях выводятся в формате - чч:мм:сс
Каждая строка журнала имеет структуру - 2025-07-10 14:23:27: Сообщение...
Файл журнала дописывается а не удаляется при каждом запуске программы

6. Основная логика работы результирующей программы
Вначале считывается и обрабатывается файл параметров EPGi.ini и актуальный файл EPG.
Затем происходит интерактивный процес предоставления информации пользователю.
Основной экран (экран1) - таблица по N строк на экране, где:
1 строка - 1 канал с актуальной (текущей) передачей идущей сейчас на канале.
Колонки (фиксированного размера):
- наименование канала
- наименование текущей передачи
- графический индикатор, показывающий % времени прошедший с начала передачи.
Экран постранично пользователь может листать.
Пользователь может также перемещаться построчно.
Нажатие Enter на строке раскрывает экран2, на который выводится детальная информация о текущей программе передач.
По Esc из экрана2 происходит возврат на экран1 в тоже строку.
По Esc из экрана1 происходит завершение работы программы.
